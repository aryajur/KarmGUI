KarmConfig.lua
This is a Lua script file which allows Karm to be customized by the user. The file is executed after the GUI is initialized so the user can
  1. Add/remove stuff from the GUI
  2. Modify the menu or the toolbar to add custom commands
  3. Load plugins
  4. Load Task Spore files
  5. Load a default filter

A default configuration file is provided in the installation package but Karm would work just fine without it and revert to defaults coded in the core code
